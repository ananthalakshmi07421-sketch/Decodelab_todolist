# 📝 To-Do List Application (Python)

A simple command-line To-Do List application built with Python. This project allows users to manage their daily tasks by adding, viewing, and deleting tasks through an interactive menu.

## 📌 Features

- ➕ Add new tasks
- 📋 View all tasks
- ❌ Delete tasks by task number
- 🔄 Interactive menu-driven interface
- ⚠ Basic error handling for invalid input
- 👋 Exit the application safely

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

### 2. Go to the project folder

```bash
cd your-repository-name
```

### 3. Run the application

```bash
python todo.py
```

> Replace `todo.py` with your actual Python filename if it is different.
